# Java: criando sua primeira aplicação

Este projeto foi desenvolvido durante o curso **"Java: criando sua primeira aplicação"** da Alura.  
O objetivo é aplicar os conceitos iniciais da linguagem Java por meio de exercícios práticos, simulando situações do dia a dia como leitura de dados, operações matemáticas e lógica de programação.

## 📚 Conteúdos abordados

Durante o desenvolvimento do projeto, foram trabalhados conceitos fundamentais como:

- Entrada de dados com `Scanner`
- Variáveis e tipos primitivos
- Operadores lógicos (`&&`, `||`)
- Operadores relacionais (`>`, `<`, `>=`, `<=`, `==`)
- Operadores de atribuição (`+=`, `-=`)
- Estruturas condicionais (`if`, `else if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Manipulação de dados e cálculos básicos
